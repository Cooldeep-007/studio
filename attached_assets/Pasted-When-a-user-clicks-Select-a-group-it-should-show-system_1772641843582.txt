🏗️ Functional Requirements:
1️⃣ Database Structure

Create a parent_groups table:

id (Primary Key)
group_name (VARCHAR, unique)
primary_nature (Assets / Liabilities / Income / Expense)
is_system (BOOLEAN default TRUE)
created_at

And update ledgers table:

parent_group_id (Foreign Key → parent_groups.id)

3️⃣ Frontend Behavior (Create Ledger Form)

The Parent Group dropdown should:

Fetch data from parent_groups table

Show only active groups

Be searchable

When user selects a Parent Group:

Auto-fill the Primary Nature field (readonly)

4️⃣ Restrictions

System groups (is_system = true) cannot be deleted.

User-created groups (is_system = false) can be edited or deleted.

Add ability to create custom parent groups.

Add hierarchy support (parent_group_id reference inside parent_groups table).
